# Minesweeper

簡易 Windows Form 踩地雷應用程式專案

### 遊戲等級

難度 | 方塊 | 地雷
---- | ---- | ---
初級 | 9 X 9 | 10
中級 | 16 X 16 | 40
高級 | 16 X 30 | 99

### 使用方法

* Clone the repository  `git clone https://github.com/hezhengmin/Minesweeper.git` 
* Open the Visual Studio Solution(.Sln) file `Minesweeper.sln`
